# mbta-prediction
A project of EC500 (cloud computing), this is a web application that aims to provide accurate prediction to users of Massacheustes Bay Transportation Authority train network. The application will predict estimated time of arrival and commute afer taking into account the various effecting factors.  

RestAPI
In branch RestAPI, the file messenger-2 is our Restful API it can be opened in Eclipse luna IDE
Our java SE version is 1.7.0_79.
By extracting the file as .war file, we can deploy it on AWS Elastic Beanstalk, the Configuration is 64bit Amazon Linux 2015.09 v2.0.8 running Tomcat 8 Java 8. It provides us the URL to get access to our database.
